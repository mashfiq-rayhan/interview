## ðŸŸ¢ Basic

### 1. What is React Native, and how does it differ from React?

React Native is a framework for building native mobile apps using React.

| Feature   | React       | React Native                   |
| --------- | ----------- | ------------------------------ |
| Platform  | Web         | Mobile (iOS/Android)           |
| UI        | HTML/CSS    | Native components (View, Text) |
| Styling   | CSS         | StyleSheet API                 |
| Rendering | Browser DOM | Native UI rendering            |

---

### 2. Core Components

**Basic Components:** `View`, `Text`, `Image`, `ScrollView`, `FlatList`, `Button`, `TextInput`

```jsx
import { View, Text } from 'react-native';

function App() {
  return (
    <View>
      <Text>Hello, React Native!</Text>
    </View>
  );
}
```

---

### 3. Props and State

**Props:** Immutable, passed from parent.
**State:** Mutable, managed via `useState`.

```jsx
function Counter({ initial }) {
  const [count, setCount] = useState(initial);
  return <Text onPress={() => setCount(count + 1)}>{count}</Text>;
}
```

---

### 4. JSX in React Native

```jsx
<View style={{ flex: 1 }}><Text>Content</Text></View>
```

---

### 5. Styling

```jsx
import { StyleSheet } from 'react-native';
const styles = StyleSheet.create({
  container: { flex: 1, backgroundColor: '#fff' }
});
```

---

### 6. Expo

Toolchain for faster development with hardware APIs.

```bash
expo init my-app
```

---

### 7. Functional vs Class Components

Prefer functional components with hooks.

---

### 8. Navigation

Use `react-navigation`.

```jsx
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
const Stack = createStackNavigator();
```

---

### 9. Flexbox

Default direction: **column**. Use `flexDirection`, `justifyContent`, `alignItems`.

---

### 10. Running the App

```bash
npx react-native run-android
npx react-native run-ios
```
